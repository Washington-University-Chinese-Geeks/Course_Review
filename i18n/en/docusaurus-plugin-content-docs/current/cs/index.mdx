# Computer Science (CSE)

All undergraduate courses from WashU's Department of Computer Science & Engineering. Cards with a **has review** badge link to a page with student reviews; the rest are awaiting a first review — open a PR to add one!

> **Course-number change**: WashU recently migrated 3-digit course numbers to 4-digit ones (e.g., CSE 131 → CSE 1301, CSE 422 → CSE 4202). Old numbers are listed on each course card when known.
